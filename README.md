# Future Ice Avalanches

## Installation
1. Clone the repository
```bash
git clone https://github.com/dav1dclara/future-ice-avalanches.git
cd future-ice-avalanches
```
2. Create virtual environment and install dependencies
```bash
conda create -n FIA python=3.11
conda activate FIA
pip install -r requirements.txt
pip install -e .
```
